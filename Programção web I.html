Programação Web I

Na disciplina de Programação Web 1, os alunos serão introduzidos aos princípios essenciais de desenvolvimento front-end, com ênfase em HTML, CSS e JavaScript. Eles aprenderão a criar estrutura de páginas com HTML, estilizar elementos utilizando CSS e adicionar interatividade e dinamismo com JavaScript. O curso abordará desde conceitos básicos até técnicas avançadas para construir interfaces web modernas e responsivas.
